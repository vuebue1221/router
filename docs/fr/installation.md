# Installation

## Téléchargement direct / CDN

[https://unpkg.com/vue-router@4](https://unpkg.com/vue-router@4)

<!--email_off-->

[Unpkg.com](https://unpkg.com) fournit des liens CDN basés sur npm. Le lien ci-dessus pointera toujours vers la dernière version sur npm. Vous pouvez également utiliser une version/tag spécifique via des URLs comme `https://unpkg.com/vue-router@4.0.5/dist/vue-router.global.js`.

<!--/email_off-->

## npm

```bash
npm install vue-router@4
```

## yarn

```bash
yarn add vue-router@4
```
